# Support

## Getting Help

Use the channel that fits the problem:

- bug reports: open a GitHub issue
- feature ideas: open a GitHub issue
- security concerns: follow [SECURITY.md](./SECURITY.md)
- contribution questions: read [CONTRIBUTING.md](./CONTRIBUTING.md)

## Before You Open an Issue

- check the README
- check `docs/`
- run the local verification commands
- include relevant logs, screenshots, and reproduction steps
